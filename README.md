# coc-project-student-score-analyser
this project is student score analyser that inputs marks of students from user and the displays the average class marks, highest marks in class and lowest marks in class
concept used are of c programming language such as for loops, arrays defination, arrays input, functions, using of libraries
concepts used are of maths such central tendancies(mean, median, mode, standard deviation, varience)
this can be compiled on visual studio code or any c language compiler 
can run on windows terminal by entering the number of students whose data is to be analysed, then entering the marks of students, the program will store them into an array and perform operations and display the highest marks, lowest marks, average marks of class
